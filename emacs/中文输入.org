* evil-surround 插件
记忆口诀：
*y = you(add) c = change d = delete s = surround*

🟢 Normal 模式

| 快捷键    | 含义助记                | 操作说明           | 实际效果（       | 为光标）          | 使用场景 |
|-----------+-------------------------+--------------------+------------------+-------------------+----------|
| *y s i w )* | you surround inner word | 给当前单词加括号   | word → (word)   | 快速包裹单词      |          |
| *y s i w (* | you surround inner word | 用带空格的括号包裹 | word → ( word ) | Lisp / 可读性友好 |          |
| *y s s )*   | you surround sentence   | 包裹整行           | text → (text)   | 快速包裹当前行    |          |
| *c s " '*   | change surround         | 替换包裹符号       | "word" → 'word' | 切换引号 / 括号   |          |
| *d s "*     | delete surround         | 删除外层包裹       | "word" → word   | 快速去壳          |          |

1. 总共就三种命令 ~ys cs ds~ 对应添加，修改，删除
2. iw in word
3. 不能选中，只能在normal 模式下自动修改行后面的部分
4. 左侧括号会加空格，右侧紧贴
