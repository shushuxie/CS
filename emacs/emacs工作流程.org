
* 工作流程
1. 打开软件，terminal
2. 查看todo
3. 做工作
   - 日记
   - 日志
   - 知乎收藏
   - bilibili收藏
   - 其他
4. 添加capture，查看agenda视图
    
  
* 常用的操作方式

1. todo添加
   * *,at* 查看todo等方式
   * *,ag* git操作
     * F fetch拉取
     * P push
2. org特殊格式
   * **text** 加粗
   * *~text~*  ~text~
   * *_text_* _下划线_
3. 其他快捷方式
   * *,<* 插入模板，latex，src等
   * *,|* table
4. 文件搜索
   新建文件treemacs
   新建目录treemacs
   搜索文件 projectial *,pf ,pd ,pg ,pr*
   pd是搜索文件夹，可以打开文件夹视图进行文件跳转
